# 에스티 설명

### 에스티는 블루투스, 시리얼 통신을 활용한 대형 조명 컨트롤러 앱입니다.

크게 조명탭, 승하강, 설정 메뉴가 있습니다.

<img width="40%" height="40%" src="https://user-images.githubusercontent.com/38831314/110316429-100ca280-804e-11eb-8e55-9670d2323d93.jpg">

첫번째로 우측 상단에 [블루투스]를 눌러서 해당 디바이스를 연결합니다.

<img width="40%" height="40%" src="https://user-images.githubusercontent.com/38831314/110316899-b9ec2f00-804e-11eb-9a46-0cffb15c551b.jpg">

해당 버튼을 클릭하면 번호가 활성화되며 메뉴 맞게 작동합니다.

<img width="40%" height="40%" src="https://user-images.githubusercontent.com/38831314/110316686-65e14a80-804e-11eb-94ed-9fe19c5500d5.jpg">

중간쯤에 위치한 전체선택, 전체해체를 통해 많은 조명을 한번에 컨트롤 할 수 있습니다.

<img width="40%" height="40%" src="https://user-images.githubusercontent.com/38831314/110317058-f029ae80-804e-11eb-8f19-a0f7cd3f33ac.jpg">

상단에 [조명설정]을 누르면 원하는 갯수만큼의 조명컨트롤이 가능합니다.

<img width="40%" height="40%" src="https://user-images.githubusercontent.com/38831314/110317272-41d23900-804f-11eb-940d-0e6a2d3b2bc3.jpg">

조명 최대 개수를 20개로 하고 A그룹을 선택한 화면입니다.

<img width="40%" height="40%" src="https://user-images.githubusercontent.com/38831314/110317958-392e3280-8050-11eb-9466-78faea5a9981.jpg">

그룹 화면에서 A ~ H 까지 조명 그룹을 선택 및 해체할 수 있습니다.(조명은 중복이 불가능합니다.)

<img width="40%" height="40%" src="https://user-images.githubusercontent.com/38831314/110318197-8ad6bd00-8050-11eb-9164-66d7177a6466.jpg">
<img width="40%" height="40%" src="https://user-images.githubusercontent.com/38831314/110318393-c6718700-8050-11eb-86b8-f7aabce14978.jpg">
<img width="40%" height="40%" src="https://user-images.githubusercontent.com/38831314/110318505-ea34cd00-8050-11eb-8e7a-74d6b0be5a84.jpg">

그룹을 활용하여 에스티를 더욱 활용할 수 있습니다.
