# 에스티 설명

### 에스티는 대형 조명 컨트롤러 앱입니다.

크게 조명탭, 승하강, 설정 메뉴가 있습니다.

<img width="40%" height="40%" src="https://user-images.githubusercontent.com/38831314/110316429-100ca280-804e-11eb-8e55-9670d2323d93.jpg">

첫번쨰로 우측 상단에 [블루투스]를 눌러서 해당 디바이스를 연결합니다.

<img width="40%" height="40%" src="https://user-images.githubusercontent.com/38831314/110316899-b9ec2f00-804e-11eb-9a46-0cffb15c551b.jpg">

해당 버튼을 클릭하면 번호가 활성화되며 메뉴 맞게 작동합니다.

<img width="40%" height="40%" src="https://user-images.githubusercontent.com/38831314/110316686-65e14a80-804e-11eb-94ed-9fe19c5500d5.jpg">

중간쯤에 위치한 전체선택, 전체해체를 통해 많은 조명을 한번에 컨트롤 할 수 있습니다.

<img width="40%" height="40%" src="https://user-images.githubusercontent.com/38831314/110317058-f029ae80-804e-11eb-8f19-a0f7cd3f33ac.jpg">
